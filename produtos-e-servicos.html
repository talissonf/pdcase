Produtos e Serviços

O módulo de “Produtos & Serviços” congrega numa base única a identificação dos ofertas a clientes, podendo manter os dados comuns, ou especificos conforme área destino de uso.


Dados Comuns das Informações

Na determinação dos dados comuns enquadram informações de uso comercial, periodo de comercialização, norma reguladora, Unidade Gestora, Cliente Foco, obrigatoriedade de uso de Contas Correntes, tendo em vista de se tratar no principal produto de relacionamento com Clientela.


Dados Distintos das Informações

Na distinção de informações são definidos dados de produtos de Investimento, de Crédito e Serviços Oferecidos. Nesse contexto são tratados limites operacionais de valores, prazos de vigencia, custos de realização, canais de relacionamento, dentre outros fatores.


Dados Específicos das Informações

Os dados especificos, tais como taxa, valores e tarifas são preferenciamente mantidos nos sistemas de negócios.
